Data Science: What is it?

  • As data proliferate (mega, giga, tera, peta, exa, zetta, yotta), individuals capable of managing and extracting meaning from data are key.
  • Data Scientist: Knows more statistics than a computer scientist and more computer science than a statistician.
  • Data "products" as opposed to data analysis.
  • Median Salary of $117,500.
  • Graduate programs in Data Science at Berkeley and NYU.

What's changed?

    Commodity hardware

    • Processing power is huge compared to just 5 years ago.
    • Computer memory and storage is immense compared to just 5 years ago.
    • Cloud computing/Platform as a Service (PaaS) providers (e.g., Amazon Web Services) lower access/cost barriers to entry.
    • Computers (virtual machines) can be rented eliminating capital costs.
Commodity hardware tied to infrastructure as a service providers like Amazon Web Services

What's changed?

    Open Source Software

    • A model of software development and distribution distinguished from proprietary software.
    • Includes operating systems like Linux/Unix/Android as well as application software like R.
    • Inexpensive (free).
    • Business models focus on building services around software.
    • Communities rapidly emerging around the development and distribution of open source software (e.g., GitHub).
Open source software allowing rapid and inexpensive (free) innovation

What's changed?

    The World Wide Web

    • Fundamentally alters the way in which people communicate and collaborate.
    • Research papers become research websites.
    • Now possible to rapidly co-develop software and share data (even confidential data) used for data analysis.
    • A very small team can now do what previously took an army of researchers.
The World Wide Web transforms modes of communication and collaboration.

Goals for Today

  • Create a GitHub account, clone a repository.
  • Install R and Rtools (Windows) on your computer.
  • Write a simple R package and commit to GitHub.
  • Install package from GitHub both locally and in the cloud/AWS.
  • Create GitHub page for project.

Let's get organized: Version Control

  • Data analysis is often a solitary endeavor.
  • Keeping track of what one is doing is usually idiosyncratic.
  • Software developers created version control systems to allow for teams to build software together well.
  • Subversion and Git (created by Linux inventor Linus Torvalds) are two of the best known version control software.

GitHub

  • GitHub is a web-based version of Git that allows for hosting of code repositories.
  • GitHub mantra: Collaboration without coordination.
  • Used for more than just software development.
  • Has allowed Adam and I to work together with dozens of states on SGP analyses.
  • Data analysts often have their own personal way of doing version control (file naming conventions).

GitHub

  • Besides version control, GitHub provides versioning statistics as well as FREE static website hosting for each project.
  • Version control and project hosting all in one!
  • Netflix develops and showcases it tools on GitHub.
  • Can also forward to alternate domain names.

GitHub for Data

R

  • R is an object oriented, open source software environment used for statistical computing and data visualization.
  • Started as a dialect of the S programming language in 1993 by Ross Ihaka and Robert Gentleman.
  • Competes with proprietary software analysis tools like SAS, SPSS, Stata.
  • R is highly extensible via packages that are easy to install from CRAN or GitHub.

Cloud Computing/Amazon Web Services (AWS)

  • Cloud computing platforms allow users access to remote computing services.
  • Amazon Web Services is the most prominent example of such a platform (Platform as a Service/PaaS).
  • Provides dozens of services.
  • For data analysis, high-end virtual machines are available for cluster computing (e.g., AWS/EC2, elastic cloud computing).
